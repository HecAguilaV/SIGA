# SIGA-APP: Movilidad Agéntica

Extensión para Android (Jetpack Compose).

##  Operación en Terreno
- **Propósito**: Facilitar la gestión de stock y ventas mediante asistencia inteligente en dispositivos móviles.
- **IA en el Dispositivo**: Planeado uso de reconocimiento de voz para dictar inventarios y consultas rápidas sobre ubicación de productos y niveles críticos de stock.

##  Integración
- Se comunica con `SIGA-BACKEND` compartiendo el mismo contexto agéntico y reglas de negocio para mantener la consistencia entre la Web y la App.
