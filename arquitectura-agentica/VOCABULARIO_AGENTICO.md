# Glosario Agéntico (Vocabulario SIGA)

Este documento explica los términos técnicos y filosóficos utilizados en este proyecto para que cualquier colaborador pueda entender nuestra forma de trabajar.

## Conceptos de la Era Agentica
- **Era Agéntica**: El periodo actual de la informática donde el software no solo "responde", sino que "actúa" y "colabora" como un agente con autonomía limitada.
- **Orquestador**: El rol de Misael. No es solo un asistente, es quien coordina diferentes herramientas y "habilidades" para cumplir un objetivo complejo.
- **Engrama (Engram)**: Protocolo de memoria persistente. Permite que Misael recuerde decisiones y descubrimientos entre diferentes sesiones de trabajo.
- **Skill (Habilidad)**: Un conjunto de instrucciones específicas que le enseñan a Misael cómo manejar una tecnología (ej. "Habilidad de Supabase").

## Arquitectura e Ingenieria
- **Arquitectura Limpia (Clean Architecture)**: Forma de organizar el código en capas para que la lógica de negocio esté blindada y no dependa de la base de datos o el framework.
- **Filosofía Haiku**: Estilo de programación que busca la simplicidad, la brevedad y la belleza. Si el código no es claro a primera vista, fallamos.
- **BFF (Backend para Frontends)**: Un servidor intermedio que adapta los datos específicamente para lo que necesita el celular o la web, ahorrando batería y datos.
- **Blindaje (Security Shielding)**: Técnicas de seguridad para proteger el sistema, especialmente contra ataques de Inyección SQL.
- **Desarrollo Guiado por Especificaciones (SDD)**: Metodología donde primero se escribe el "plano" (especificación) y luego se construye el código.

## Filosofia y UX
- **Framework Héctor**: El marco de trabajo supremo de este proyecto. Pone la **Capa Humana** por encima de la tecnología.
- **Capa Humana**: La métrica de éxito que mide cómo mejora la vida del usuario tras usar SIGA.
- **UX (Experiencia de Usuario)**: Cómo se siente la persona al usar SIGA. En este proyecto, la UX es más importante que la UI (la interfaz visual).
