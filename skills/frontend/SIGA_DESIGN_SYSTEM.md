# Gentle Design System (SIGA)

Este protocolo guía la creación de interfaces que "enamoran" en SIGA - Gentle Nexus.

## 🎨 Principios Estéticos (Anthropic Style)
- **Nada de "AI Slop"**: Evitar tipografías genéricas (Arial, Inter básica). Usar fuentes con carácter (ej. `Space Grotesk`, `Outfit` o fuentes de revista).
- **Contraste Audaz**: Usar paletas coherentes pero con acentos fuertes. Nada de degradados morados sobre blanco.
- **Espacio**: Layouts asimétricos y generosos. Romper la cuadrícula en puntos estratégicos.
- **Motion**: Micro-animaciones CSS-only para feedback inmediato. Animaciones de carga escalonadas.

## ⚙️ Rigor Técnico (Vercel Style)
- **Accesibilidad**: ARIA labels, contraste de color WCAG, navegación por teclado.
- **Rendimiento**: Cero placeholders. Imágenes optimizadas. Componentes reutilizables como módulos NPM.
- **BFF-Oriented**: Cada componente debe pedir solo los datos que consume.

## 🛡️ Reglas de Oro
1.  Si se ve aburrido, cámbialo.
2.  Si es lento, optimízalo.
3.  Si no es accesible, no existe.
